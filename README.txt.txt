* رزومه آنلاین*

: هدف
- نمایش اطلاعات حرفه‌ای .
- تمرین انتشار پروژه‌های  با GitHub Pages.

: نحوه اجرا
- برای مشاهده رزومه، به آدرس [GitHub Pages](https://username.github.io/repo-name) مراجعه کنید.
- برای توسعه محلی، کافی است فایل‌های `index.html` و `style.css` را ویرایش کنید.

: فایل‌های پروژه
- `index.html`: ساختار اصلی رزومه.
- `style.css`: استایل‌های صفحه.
